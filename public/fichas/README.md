# Fichas de Modelos en PDF

Esta carpeta contiene las fichas técnicas en PDF de los modelos de autobuses.

## Archivos Requeridos

Coloca los siguientes archivos PDF en esta carpeta:

1. `ficha-modelo-1.pdf` - Ficha del Modelo 1
2. `ficha-modelo-2.pdf` - Ficha del Modelo 2
3. `ficha-modelo-3.pdf` - Ficha del Modelo 3
4. `ficha-modelo-4.pdf` - Ficha del Modelo 4

## Cómo Funciona

Cuando un usuario hace clic en uno de los botones "MODELO X" en la página de inicio, se descargará automáticamente el PDF correspondiente.

## Personalización

Si deseas cambiar los nombres de los botones o agregar más modelos, edita el archivo:
`src/pages/LandingPage.jsx`

Busca la sección con los 4 botones y modifica según tus necesidades.
